Software Design Document (SDD)

Course: CP-317-C, Wilfrid Laurier University

1. Introduction

This project processes two text files—NameFile.txt and CourseFile.txt—to generate a summarized performance report for students. It calculates final grades using the formula: 

Final Grade = Test1 * 0.2 + Test2 * 0.2 + Test3 * 0.2 + FinalExam * 0.4

The output is written to a new file, sorted by student ID. This group project emphasizes object-oriented programming and collaboration.

2. Architecture Design

Architecture: Monolithic

All features are integrated into a single Java program. It performs file I/O, data processing, sorting, and output generation.

2.1 System Components

  • Input Module: Reads and validates NameFile.txt and CourseFile.txt
  • Data Processing Module: Matches student IDs and calculates grades
  • Sorting Module: Sorts students by ID using Collections.sort()
  • Output Module: Formats and writes output to StudentDatabase.txt

3. Detailed Design

3.1 Overview

Written in Java, this program reads two files, computes final grades, and outputs a sorted table.

3.2 Grade Calculation

Each line in CourseFile.txt is split by commas. The last four values represent Test1, Test2, Test3, and Final Exam scores.

3.3 Functions

The following functions are used:

  • readNameFile() – Parses student names and IDs
  • readCourseFile() – Parses test and course data
  • processData() – Merges and calculates final grades
  • sortData() – Sorts by student ID
  • writeOutputFile() – Outputs formatted table
